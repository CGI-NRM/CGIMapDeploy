# CGIMapDeploy
In order to deploy a map; branch this repository, clone it down, add your data, generate and commit.

If you deploy a qpcr map, it will be reachable from https://cgi-nrm.github.io/[repository name]/QPCR_Map.html

(Link to example map: https://cgi-nrm.github.io/CGIMapDeploy/QPCR_Map.html)
