# CGIMapDeploy
In order to deploy a map; fork this repository, clone it down, add your data, generate and commit.

You will also have to go in under settings on the github website for the repository and deploy the map:
Settings -> Pages -> Branch (change 'None' to 'main')

If you deploy a qpcr map, it will be reachable from `https://cgi-nrm.github.io/[repository name]/QPCR_Map.html`.

(Link to example map: https://cgi-nrm.github.io/CGIMapDeploy/QPCR_Map.html)
